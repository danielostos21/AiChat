Chatbot Developed bade on ChatGpt 3.5-turbo connected by Open Ai Api. Django Web Application.

Features:
    - Login Authentication
    - Chatbot 
    - Ability to draft messages and create content
    - Ability to answer questions
    - Ability to engage in casual conversations
    - Ability to generate code for developers and data scientists
    - Ability to fix bugs in code (in generic cases)
